# <center> **Recommender Systems Projects**

Collection of my projects and experiments in **Recommender Systems** — from classical approaches to modern deep learning architectures.

---

### 📁 Projects

- **[Book Recommendation Service](Book-Recommendation-System/)**  
  End-to-end hybrid recommender for books combining classical methods (Popularity, Content-based, Item-Based CF, Matrix Factorization) with a neural **Wide & Deep** model. Achieved significant improvement in ranking metrics.


---

### 🛠 Technologies & Approaches

**Classical Methods**
- Popularity Baseline
- Content-based Filtering
- Collaborative Filtering (Item-Based)
- Matrix Factorization (SVD)

**Advanced & Neural Approaches**
- Wide & Deep architecture
- Two-Tower models (planned)
- Hybrid ensembles
- Candidate generation + reranking pipelines

**Evaluation**
- Precision@K, Recall@K, nDCG@K, MAP@K
- Offline metrics + user segmentation analysis

**Tools**
- Python, Pandas, NumPy
- scikit-learn, Surprise, Implicit
- PyTorch (for neural models)
- Jupyter Notebooks

---

### 🎯 Goals of the Repository

- Compare effectiveness of different recommendation paradigms
- Build production-like hybrid systems
- Experiment with modern deep learning recommenders
- Improve metrics through ensembling and neural architectures
